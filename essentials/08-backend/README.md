# Step

## 1. init  
Init 명령으로 Terraform 수행을 위한 provider plugin 초기화 및 다운로드 수행

```
terraform init
```

## 2. plan  
Plan 명령으로 Terraform 수행 전 실행 시뮬레이션 확인
```
terraform plan
```  

## 3. apply  
Apply 명령으로 Terraform 을 통한 Resource 생성 수행
```
terraform apply
```  

## 4. 실행 내용 확인
선택한 Region 에 VPC 생성 내용 확인 


# Resource 삭제
